
📄 INSTRUCCIONES PARA USAR TU LANDING PAGE - RIMAN

Este paquete contiene todo lo necesario para visualizar y publicar tu landing page de productos de belleza coreana.

───────────────────────────────────────────
🔧 OPCIÓN 1: VERLO EN TU COMPUTADOR (CON VS CODE)
───────────────────────────────────────────

1. Asegúrate de tener Visual Studio Code instalado.
2. Abre la carpeta RIMAN_LandingPage_Completo.zip y extrae su contenido.
3. Instala la extensión "Live Server" desde la tienda de extensiones de VS Code.
4. Abre la carpeta extraída desde VS Code.
5. Haz clic derecho en el archivo index.html y selecciona "Open with Live Server".
6. ¡Tu landing se abrirá en el navegador!

───────────────────────────────────────────
🌐 OPCIÓN 2: PUBLICAR EN GITHUB PAGES (GRATIS)
───────────────────────────────────────────

1. Entra a https://github.com y crea un nuevo repositorio (por ejemplo: riman-landing).
2. Sube los archivos del .zip (NO subas el zip, solo su contenido).
3. Ve a la configuración del repositorio → Pages.
4. En "Branch", selecciona main y la carpeta raíz.
5. Se generará un enlace público para tu landing (ej. https://tuusuario.github.io/riman-landing).
6. ¡Listo! Puedes compartirlo con tus clientes.
